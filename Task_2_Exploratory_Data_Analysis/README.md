# Task 2: Exploratory Data Analysis (EDA)

## Overview
This project focuses on performing Exploratory Data Analysis (EDA) on a sales dataset to identify patterns, trends, and relationships within the data. The goal is to gain meaningful insights into customer purchasing behavior and business performance through statistical analysis and data visualization.

## Project Environment
The analysis was performed using a Kaggle Notebook environment with Python and popular data analysis libraries.

## Tools and Technologies Used
- Kaggle Notebook
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset Features
The dataset includes:
- Quantity
- UnitPrice
- ItemsInCart
- TotalPrice
- Transaction Date

## Analysis Performed
- Data Exploration
- Statistical Summary
- Total Price Distribution Analysis
- Outlier Detection using IQR Method
- Monthly Sales Trend Analysis
- Quantity Distribution Analysis
- Correlation Analysis using Heatmap

## Key Findings
- Most transactions fall within the low to medium spending range.
- A small number of high-value transactions contribute significantly to overall revenue.
- Monthly sales trends indicate fluctuations in customer demand over time.
- Quantity and TotalPrice show a strong positive relationship.
- Quantity and ItemsInCart are positively correlated.
- UnitPrice has minimal influence on purchase quantity.

## Business Insights
- Revenue growth is primarily driven by purchase volume rather than product pricing.
- Increasing cart size can improve overall sales performance.
- Bundle offers and product recommendations can encourage larger purchases.
- High-value customers can be targeted through loyalty programs and personalized promotions.

## Conclusion
The exploratory data analysis revealed important patterns in sales performance and customer purchasing behavior. The findings indicate that purchase quantity and cart size are major contributors to revenue growth. By leveraging these insights, businesses can make data-driven decisions to improve sales strategies, customer engagement, and overall business performance.
